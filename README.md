# 🌱 Sustainable Agribusiness Optimisation (NAB Capstone)

## 🔍 Overview
This project was developed in collaboration with National Australia Bank (NAB) to design a data-driven sustainability solution for an agribusiness. The objective was to create a viable green initiative supported by structured analysis and project planning.

---

## 🎯 Objectives
- Develop a sustainability-focused business solution  
- Support decision-making using real-world data  
- Create a full Project Management Plan (PMP)  
- Align with NAB green financing requirements  

---

## 🛠 Tools & Technologies
- Excel (analysis and modelling)  
- Project Management frameworks (PMP)  
- Climate and agricultural datasets  

---

## ⚙️ Key Work

- Designed a **biogas-based anaerobic digestion system** to reduce emissions  
- Developed a full **Project Management Plan (PMP)**  
- Applied **Waterfall methodology** for project execution  
- Built a **risk management framework (RPN scoring)**  
- Created stakeholder analysis (RACI matrix, Power–Interest grid)  

---

## 📊 Outcomes
- Delivered a structured sustainability solution  
- Demonstrated environmental and financial impact  
- Simulated real-world stakeholder decision-making  

---

## 🏆 Achievement
- 🥇 Awarded **Best Capstone Project**

---

## 🚀 Conclusion
This project demonstrates the ability to combine business analytics, data-driven reasoning, and structured project management in a real-world consulting scenario.
